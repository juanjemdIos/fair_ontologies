FOOPS! is available at: https://w3id.org/foops/

Authors: Daniel Garijo and María Poveda, with contributions from Jacobo Mata and Juan Jesús Mendoza.

FOOPS! is an application for assessing whether a vocabulary (OWL or SKOS) conforms with the FAIR data principles.

Our ISWC 2021 demo paper (best demo award) provides an overview of the FOOPS! service.
